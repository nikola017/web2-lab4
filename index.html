Demonstracija HTTP/2 Učinkovitosti

Ovaj projekt demonstrira brže učitavanje vlastite web stranice korištenjem HTTP/2 protokola. U prvom dijelu projekta smo instalirali web poslužitelj nginx, izradili vlastiti SSL certifikat, aktivirali HTTPS/SSL, konfigurirali podršku za HTTP/2 protokol u nginx poslužitelju, i uključili HTTP/2 protokol. Nakon toga, potvrdili smo da je podrška za HTTP/2 protokol uspješno uključena.

U drugom dijelu projekta smo stvorili direktorij "lab4" u html direktoriju poslužitelja i izradili vlastitu web stranicu koja sadržava tekst i veći broj CSS ili JS datoteka, te više pojedinačnih datoteka slika (JPEG, PNG, svakako više od 10).
